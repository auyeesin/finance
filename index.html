1. Wrong URL format

2. Sheet not published
In Google Sheets: File → Share → Publish to web

3. Published the wrong sheet
Make sure you published Transactions specifically — not "Entire Document".

4. Network/browser blocking it
Try opening your dashboard in Chrome. Some browsers block cross-origin requests from local HTML files. In Chrome, try opening the URL directly first to confirm it shows CSV data.
